# Chinese Poetry Platform

## Project Overview

Chinese Poetry Platform is a web-based educational platform designed to introduce Chinese classical literature, including Tang Poetry and Song Lyrics. The platform provides structured navigation, poet introductions, and literary content to help users explore traditional Chinese culture.

---

## Technologies Used

* HTML5
* CSS3
* JavaScript

---

## Features

### Literature Categorisation

* Tang Poetry Section
* Song Lyrics Section
* Famous Poet Profiles

### Content Navigation

* Multi-page website structure
* User-friendly navigation system
* Organised literary resources

### Information Presentation

* Author introductions
* Poetry appreciation content
* Traditional culture promotion
